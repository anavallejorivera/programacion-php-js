# Programación PHP y JavaScript

Un repositorio educativo con ejercicios y lecciones de un curso de programación en **PHP** y **JavaScript** en desarrollo.

## 📚 Descripción

Este proyecto contiene una serie de lecciones y ejercicios prácticos organizados por fechas, cubriendo conceptos fundamentales de:

- **PHP**: tipos de datos, manejo de fechas con Carbon, formatos de datos, presentación de información
- **HTML/JavaScript**: interactividad en el navegador
- **Desarrollo web Full Stack**: Frontend y Backend

## 🚀 Cómo Ejecutar

### Requisitos
- PHP 7.0+ instalado
- XAMPP, LAMP, LEMP o servidor web similar
- Navegador web

### Instalación
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/anavallejorivera/programacion-php-js.git
   cd programacion-php-js
   ```

2. Colocar en la carpeta `htdocs` de XAMPP (o raíz del servidor web)

3. Acceder a través del navegador:
   ```
   http://localhost/programacion-php-js/
   ```
